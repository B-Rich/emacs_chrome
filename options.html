For Edit with Emacs to work you need to be running an "edit server" on your local machine. This is because extensions in Chrome(ium) cannot directly start new programs. There are several options available to you. For Emacs users it is recommended you use the native edit-server.el from the project site. One popular way of starting it is for emacs --daemon invocations is to add the following code to your .emacs: 
 (if (and (daemonp) (locate-library "edit-server"))
     (progn
       (require 'edit-server)
       (edit-server-start)))

Other options exist including an edit server written for TextAid which can be found here.
